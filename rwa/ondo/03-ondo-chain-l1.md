# Ondo Chain — 为什么做 L1

> 文档：https://docs.ondo.finance/ondo-chain/overview
> 注意：Ondo Chain 截至调研时尚未上线

---

## 核心定位

「公开可访问 + 受许可验证节点」的 L1 区块链，专为机构级 RWA 金融市场设计。

**不是普通公链，也不是私有链。** 是两者的混合：
- 任何人都可以发行 Token、开发应用、作为用户投资
- 但验证节点必须是受许可的机构（资产管理人、持牌券商）

---

## 为什么要建 L1（官方理由）

**理由一：现有链无法满足机构需求**

Ethereum 是通用链，MEV（矿工/验证者抢跑）、匿名节点、高 gas fee 都不适合机构金融。Ondo Chain 的验证节点受监控，防止前置交易，提供"最优执行"保证。

**理由二：共识层内置 Oracle + 储备证明**

普通链上的 Oracle 是第三方协议，有延迟和单点风险。Ondo Chain 在出块时自动验证链下资产数据（价格 + 托管证明），Oracle 结果是共识产物，不可单独篡改。

**理由三：机构背书 + Token 经济**

验证节点质押的是 RWA（低波动性资产），而非 ETH（高波动性）。
ONDO Token 有了网络安全质押场景，估值叙事从「应用 Token」升级为「基础设施 Token」。

---

## 核心功能

- **RWA 质押验证**：验证节点用 RWA 资产质押保证安全，而非 Crypto
- **内置 Oracle**：资产价格和托管证明在共识层自动验证
- **原生跨链**：Ondo DVN（去中心化验证网络）支持 EVM 和非 EVM 跨链
- **应用场景**：Prime Brokerage、链上财富管理、RWA 抵押借贷

---

## 战略本质

Ondo 做 L1 不是技术选择，是商业战略：

```
AUM $10亿 → 需要 ONDO Token 有更大估值空间
做应用 → Token 估值上限受限
做 L1 → Token = 网络基础资产 → 估值 10x 空间
```

---

## 对自建 RWA 项目的启示

**现阶段不需要做 L1，原因：**

1. Ondo Chain 有 $10 亿+融资、BlackRock 背书，目前还没上线
2. 做 L1 需要开发者生态、网络效应、Token 经济，资本需求数亿美元起
3. MAS 监管框架下，Ethereum/Base 已被机构接受，做新链增加额外合规摩擦

**正确的链选择：**

- **首选：Ethereum Mainnet** — 机构信任度最高，DeFi 流动性最深，OUSG/USDY 都在这里
- **次选：Base（Coinbase L2）** — gas 成本低，DeFi 生态快速建立，Coinbase 持牌背书

**何时考虑做自己的链：**
- AUM 超过 $1 亿，需要更低 gas 成本
- 有明确的生态建设计划（吸引其他 RWA 项目部署）
- 有充足资金支持节点运营和激励
